# AI Contact Manager

## Features

- Create contacts by parsing information through emails

## Development

### Pre-requisites

- Node.js LTS v20.0.0 or higher
- pnpm
- Cloudflare account

### Technologies

- Common:
    - Node.js
    - pnpm
    - TypeScript
- Frontend:
    - React
    - Tailwind CSS
    - Remix
- Backend:
    - Hono
